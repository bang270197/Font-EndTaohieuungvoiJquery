Ý nghĩa hàm find:Giúp tìm thành phân trong thành phần cha.

Demo: 

        
            <script>
                $(document).ready(function () {
                    $('li').find('.test').css('background-color', 'blue');
                })
            </script>
               <ul>
            <li class="first">Thành phần thứ 1</li>
            <li class="second">Thành phần thứ 2</li>
            <li class="third">
                <ul>
                    <li>Thành phần con thứ 1</li>
                    <li class="test">Thành phần con thứ 2</li>
                    <li>Thành phần con thứ 3</li>
                </ul>
            </li>
            <li class="fourth">Thành phần thứ 4</li>
            <li class="fifth">Thành phần thứ 5</li>
        </ul>

Ý nghĩa hàm hide:Làm ẩn các thành phần

Demo: 
        
            <script>
                $(document).ready(function () {
                    $('button.hide').click(function () {
                        $('.test03').hide(2000);
                        $('.test02').hide('slow');
                        $('.test01').hide();
                    });
                })
            </script>
             <p>
            <button class="hide">Click</button>
        </p>
        <div class="test03">hide(2000)</div>
        <div class="test02">hide('slow')</div>
        <div class="test01">hide()</div>

Ý nghĩa:Ẩn các class

hide(2000)
hide('slow')
hide()
asddas...

Ý nghĩa hàm hide:Ẩn và hiện các thành phần

Demo: 

        
            <script>
                  $(document).ready(function () {
                      $('button.clicktoggle').click(function () {
                          $('.toggle').toggle(1000, function () {
                              $(this).css('background-color', 'red');
                          });
                      })
                  });
            </script>
            <button class="clicktoggle">Click</button>
        <div class="toggle">toggle</div>

Ý nghĩa:Ẩn và hiện các class

toggle

Ý nghĩa hàm hide:Hiện các thành phần

Demo: 

              
                  <script>
                       $(document).ready(function () {
                           $('button.clickshow').click(function () {
                               $('.show').show(1000);
                           })
                       });
                  </script>
                  <button class="clickshow">Click</button>
        <div class="show">Show()</div>

Ý nghĩa:Hiện các thành phần

Show()

Ý nghĩa hàm hide:Thực hiện một hình ảnh động (animate) tùy chỉnh của một tập hợp các thuộc tính css.

Demo: 

        
            <script>
                $(document).ready(function () {
                    $('button.clickanimate').click(function () {
                        $('img').animate({
                            opacity: 0.25,
                            left: '+=150',
                            height: 'toggle'
                        }, 2000, function () {
                            $('button').text("Click tiếp để thấy hiệu ứng");
                        });
                    })
                });
            </script>
             <button class="clickanimate">Click</button>
        <img src="img/tải%20xuống%20(4).jpg">
